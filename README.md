# Book Tracker

## Описание
Book Tracker — это графическое приложение, разработанное на Python с использованием библиотеки Tkinter. Приложение предназначено для управления списком прочитанных книг. Оно позволяет пользователям добавлять информацию о книгах, фильтровать список по жанру и количеству страниц, а также сохранять и загружать данные в формате JSON.

## Функциональность
- Добавление новой книги с полями: название, автор, жанр, количество страниц.
- Фильтрация книг по жанру.
- Фильтрация книг по количеству страниц.
- Сохранение данных о книгах в формате JSON.
- Загрузка данных из файла JSON при запуске приложения.
- Проверка ввода данных для обеспечения корректности.
